# CI/CD Pipeline Practice

## What is CI/CD?
- CI: Automated code reviewer — Does this break the app?
- CD: Automated delivery — Ship to production without manual steps

## Pipeline Formula
Build → Test → Ship

## GitHub Actions Hierarchy
Workflow → Jobs → Steps

## Key Concepts
- YAML syntax for pipeline config
- Runner: Fresh cloud VM for each build
- Secrets: Never hardcode API keys
- Green checkmark = successful pipeline
